# SeriesViewer

Desktop medical image viewer built with Qt and C++.

## Features
- Load and display DICOM images
- Load and display MHA images
- View DICOM tags
- Histogram analysis for MHA

## Tech Stack
- C++
- Qt
- DCMTK
- ITK
- VTK

## Project Structure
- src: source files
- include: headers
- ui: Qt UI
- resources: resource files
